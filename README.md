# 🎓 Smart Campus AI

Smart Campus AI is a web-based platform designed to help colleges manage classrooms, labs, auditoriums, and schedules more efficiently.

## 🚀 Features

- 🏫 College and block configuration
- 🗺️ Interactive campus blueprint editor
- 📚 Classroom, lab, seminar hall, and auditorium management
- 📅 Timetable scheduling system
- 🤖 AI-based conflict detection for schedules
- 🔍 Smart search for rooms, professors, subjects, and sections
- 📊 Dashboard with utilization insights
- 🌐 3D campus visualization
- ⚡ Real-time room status tracking

## 💡 Why This Project?

Managing rooms and schedules in large campuses can be difficult and time-consuming. Smart Campus AI helps administrators organize resources efficiently, avoid timetable conflicts, and monitor campus usage through an interactive interface.

## 🛠️ Technologies Used
